# 服务商说明文档

## 服务商模式

服务商模式下，一个APPID下可绑定多个mchid，一个mchid可绑定3个APPID。

```
信息流
+-------------+         +------------------------+
|             |         |         <----->[商户1]  |
| [Wexin Pay] |<------->|  [服务商]<----->[商户2]  |
|             |         |         <----->[商户3]  |
+-------------+         +------------------------+

资金流
+-------------+         +------------------------+
|             |<--------->[商户1]                 |
| [Wexin Pay] |<--------->[商户2]        [服务商]  |
|             |<--------->[商户3]                 |
+-------------+         +------------------------+
```
